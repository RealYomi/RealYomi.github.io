Covariance Matrices in Computational Mechanics: From Spatial Variability to Eigenmodes

Covariance Matrices in Computational Mechanics:
From Spatial Variability to Eigenmodes

Spatial variability is an unavoidable feature of real engineering materials.
Whether we are dealing with concrete, rock, biological tissue, or additively manufactured metals, the Young's modulus field is rarely uniform. Instead, it fluctuates across space due to microstructural heterogeneity, defects, porosity, or manufacturing inconsistencies.

In computational mechanics, these fluctuations are often modeled as a random field.
To describe such a field mathematically, the central object is the covariance function:

$$ C(\mathbf{x}, \mathbf{x}') = \mathrm{Cov}[E(\mathbf{x}), E(\mathbf{x}')] $$

This function tells us how strongly the Young's modulus at point \(\mathbf{x}\) correlates with the modulus at another point \(\mathbf{x}'\).
Once discretized over a mesh, this becomes a covariance matrix, which plays a foundational role in:

  • stochastic finite element methods (SFEM)
  • Karhunen–Loève expansions (KLE)
  • principal component analysis (PCA) of material fields
  • uncertainty quantification
  • reduced-order modeling

In this post, we walk through the full workflow:

  1. building a covariance matrix from spatial coordinates
  2. using a variogram model to encode spatial correlation
  3. computing eigenvalues and eigenvectors
  4. truncating the spectrum to obtain a reduced-order representation
  5. visualizing the covariance structure and eigenmodes

All computations are performed using Python, NumPy, SciPy, and GSTools.

1. Mathematical Background

1.1 Covariance of a Random Material Field

Let \(E(\mathbf{x})\) denote the Young's modulus at location \(\mathbf{x}\).
The covariance function is defined as:

$$ C(\mathbf{x}, \mathbf{x}') = \mathbb{E}\left[(E(\mathbf{x}) - \mu)(E(\mathbf{x}') - \mu)\right] $$

For second-order stationary fields, the covariance depends only on the separation distance:

$$ C(\mathbf{x}, \mathbf{x}') = C(h), \qquad h = \|\mathbf{x} - \mathbf{x}'\| $$

This assumption is common in stochastic mechanics and simplifies modeling considerably.

1.2 Variograms and Their Relation to Covariance

A widely used tool in geostatistics and random field modeling is the variogram:

$$ \gamma(h) = \frac{1}{2}\mathbb{E}\left[(E(\mathbf{x}) - E(\mathbf{x}+h))^2\right] $$

For stationary fields, covariance and variogram are related by:

$$ C(h) = C(0) - \gamma(h) $$

In this work, we use the Rational variogram model:

$$ \gamma(h) = \sigma^2 \left[ 1 - \left(1 + \frac{h^2}{\ell^2}\right)^{-\alpha} \right] $$

where:

  • \(\sigma^2\) is the variance
  • \(\ell\) is the correlation length
  • \(\alpha\) controls smoothness

This model is flexible and widely used in random field generation.

1.3 Eigenvalue Problem and the Karhunen–Loève Expansion

Once the covariance matrix \(\mathbf{C}\) is assembled, we solve:

$$ \mathbf{C}\,\phi_i = \lambda_i \phi_i $$

The eigenpairs \((\lambda_i, \phi_i)\) have deep physical meaning:

  • \(\lambda_i\) quantifies the "energy" or contribution of mode \(i\)
  • \(\phi_i\) is a spatial pattern of variability (a mode shape of randomness)

Truncating after \(n\) modes yields a reduced-order representation:

$$ E(\mathbf{x}) \approx \mu + \sum_{i=1}^{n} \sqrt{\lambda_i}\,\phi_i(\mathbf{x})\,\xi_i $$

This is the Karhunen–Loève expansion (KLE), a cornerstone of stochastic mechanics.

Python Workflow

Below is an annotated Python script used to compute and plot the covariance matrix and its eigenmodes.

import numpy as np
import gstools as gs
import pandas as pd
import matplotlib.pyplot as plt
from scipy.spatial.distance import cdist

# Load FE mesh coordinates (x and y)
xi = np.arange(0, 19.51, 0.75)   # 27 points, 0 → 19.5
yi = np.arange(0, 41.26, 0.75)   # 57 points, 0 → 41.25

LxD = len(xi)
LyD = len(yi)

# Create full grid of coordinates
x, y = np.meshgrid(xi.values, yi.values)
x = x.ravel('F')
y = y.ravel('F')
locations = np.column_stack([x, y])
A = len(locations)

# Define variogram model for Young's modulus field
variogram_model = gs.Rational(
    dim=2, var=1.0, len_scale=3.0, alpha=1.14, nugget=0.0
)

# Compute pairwise distances
di = cdist(locations, locations)

# Evaluate variogram and covariance
gamma = variogram_model.variogram(di)
sill = variogram_model.var + variogram_model.nugget
covariance_matrix = sill - gamma

# Eigen-decomposition
eigenvalues, eigenvectors = np.linalg.eig(covariance_matrix)

# Sort and truncate eigenpairs
idx = eigenvalues.argsort()[::-1]
eigenvalues = eigenvalues[idx]
eigenvectors = eigenvectors[:, idx]

total = np.sum(eigenvalues)
target = 0.975 * total
n_terms = np.argmax(np.cumsum(eigenvalues) >= target)

4. Interpretation of Results

4.1 Covariance Matrix Structure

The covariance matrix exhibits a smooth decay away from the diagonal, reflecting the spatial correlation imposed by the Rational variogram. Points closer together in the mesh have higher covariance, as expected for a material field with finite correlation length.

4.2 Eigenvalue Decay and Truncation

The eigenvalues decay rapidly, indicating that the Young's modulus field is highly compressible in the KLE basis. Retaining only the first \(n\) modes captures 97.5% of the total variance — a common threshold in stochastic mechanics.

4.3 Eigenvector Shapes

The first few eigenvectors represent:

  • large-scale spatial fluctuations
  • smooth global modes
  • physically interpretable patterns of stiffness variation

Higher modes capture finer-scale oscillations but contribute less to the total variance.

5. Conclusion

Covariance matrices are the backbone of random field modeling in computational mechanics. By combining:

  • a variogram model
  • a discretized mesh
  • eigen-decomposition
  • truncation based on variance content

we obtain a powerful reduced-order representation of spatially varying Young's modulus fields.

This workflow is essential for:

  • stochastic finite element analysis
  • uncertainty quantification
  • reliability assessment
  • model reduction
  • simulation of heterogeneous materials

The Python implementation shown here provides a practical and extensible foundation for advanced stochastic modeling.
